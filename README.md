# Basic command Git
#### Initialzed
```
 git init
```

> git init it's means initialzed using Git in the project.
---
#### Add
```
 git add .
```
 > It's means Adding all modified files.

```
 git add [file name]
```
> It's means Adding a single modified file.
---
#### Comment
```
 git commit -m "[comment to your code]"
```
> It's means comment your code when you change into the line of code.
---
#### Branch
```
 git branch -M main
```
> It's means change the branch name.
---
#### Link to GitHub
```
 git remote add origin [your respository]
```
> It's means Link all of code to GitHub.
---
#### Update
```
 git push -u origin main
```
> It's means push code (update) to GitHub.
